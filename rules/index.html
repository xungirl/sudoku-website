How to Play Sudoku

Master the classic number puzzle

Basic Rules

Sudoku is a logic-based number placement puzzle. The objective is to fill a grid with digits so that each column, each row, and each of the subgrids contain all of the digits from 1 to N, where N is the size of the grid.

The Grid

A standard Sudoku puzzle uses a 9Ã—9 grid, divided into nine 3Ã—3 subgrids called "boxes" or "regions". Our Easy mode uses a 6Ã—6 grid with 2Ã—3 subgrids, perfect for beginners!

The Goal

Fill in every empty cell with a number. When complete, every row, column, and subgrid must contain each number exactly once, with no repetition.

Tips & Strategies

Scanning

Look across rows and columns to find where a specific number can go. If a number appears in two rows of a box, it must go in the third row.

Elimination

For each empty cell, note which numbers are already in its row, column, and box. The remaining possibilities are your candidates.

Single Candidate

If only one number is possible for a cell after elimination, that must be the answer!

Take Your Time

Sudoku rewards patience and logical thinking. Don't guess - every puzzle has a unique solution that can be found through pure logic.
